# Sahara PWA - Sequential Prompts

This folder contains 8 comprehensive prompts to build the Sahara chat interface.

## Execution Order

| # | File | Description |
|---|------|-------------|
| 1 | [01-install-components.md](./01-install-components.md) | Install shadcn components |
| 2 | [02-pwa-setup.md](./02-pwa-setup.md) | PWA manifest & meta tags |
| 3 | [03-chat-types-mock-data.md](./03-chat-types-mock-data.md) | TypeScript types & mock data |
| 4 | [04-chat-context.md](./04-chat-context.md) | React context for state |
| 5 | [05-chat-ui-components.md](./05-chat-ui-components.md) | Chat UI components |
| 6 | [06-chat-input.md](./06-chat-input.md) | Message input component |

| 8 | [08-main-page-demo.md](./08-main-page-demo.md) | Main pages & demo flow |

## How to Use

1. Open a prompt file
2. Copy the content to your AI assistant
3. Let it execute the steps
4. Move to the next prompt

## Rules

⚠️ **DO NOT** modify `globals.css` color values  
⚠️ **DO NOT** create custom components - use shadcn CLI only  
⚠️ Execute prompts **in order**
